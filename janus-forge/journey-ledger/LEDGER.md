# Journey Ledger: Master Log

**Purpose:** Persistent record of all Janus Forge iteration journeys

---

## What Gets Logged

Every completed session adds an entry here with:
- Session ID and timestamp
- Original intent
- Outcome summary
- Polyglow snapshot
- Omega-nodes extracted (if any)
- Key decision points
- Recursion depth used

---

## Ledger Format

```markdown
### [session-YYYY-MM-DD-NNN] Title

**Date:** YYYY-MM-DDTHH:MM:SSZ
**Depth:** N
**Intent:** Brief description of what we tried to do

**Outcome:** Success | Partial | Failed | Inconclusive

**Polyglow:**
- Heat: 0.XX
- Forge: 0.XX
- Clarity: 0.XX
- Divergence: 0.XX
- Safety: 0.XX
- Balance: 0.XX

**Key Decisions:**
1. Decision point 1
2. Decision point 2

**Patterns Observed:**
- Pattern 1 (link to patterns.md if applicable)
- Pattern 2

**Omega-Nodes Extracted:**
- None | omega-nodes/nodes/NODE-XXX.md

**Afterglow Emitted:**
- None | afterglow/emissions/EMISSION-XXX.md

**Notes:**
Free-form observations, learnings, or questions.

---
```
