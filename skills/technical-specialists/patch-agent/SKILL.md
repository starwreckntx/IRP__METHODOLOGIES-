## Profile

**Primary Role:** Patch application execution and verification

**Capabilities:**
- Patch deployment
- Application verification
- Rollback capability
- Testing coordination
- Status tracking

---

## Patch Workflow

1. **Receive** - Get patch directive
2. **Validate** - Verify patch integrity
3. **Apply** - Execute patch deployment
4. **Verify** - Confirm successful application
5. **Report** - Update status and metrics
